# CytologyTrainer — Образовательная платформа по клинической цитологии

## Описание продукта

Образовательная платформа по клинической цитологии для русскоязычных медицинских специалистов.
Обучает гинекологической цитологии (система Bethesda) и цитологии щитовидной железы через
записанные видеокурсы, живые Zoom-занятия, тесты в Telegram и практику в Ереване.

---

## Целевая аудитория

- Врачи-лаборанты и ординаторы без опыта в цитологии
- Начинающие цитологи, которые учились хаотично, без системы
- Выпускники медвузов, ещё не выбравшие специализацию
- Русскоязычные специалисты за пределами Армении
- Жители Армении вне Еревана
- Те, кто готов приехать на офлайн-практику в Ереван

---

## Проблема

В русскоязычном пространстве нет структурированного обучения цитологии по международным стандартам.
Начинающие специалисты учатся хаотично, без системы и практической базы.
Качественное обучение недоступно за пределами крупных городов.

**Главное конкурентное преимущество:** прямых русскоязычных конкурентов по цитологии практически нет.

## Технический стек

| Компонент | Инструмент | Описание |
|-----------|-----------|----------|
| Сайт/лендинг | Tilda | Без кода, быстро, красиво |
| Видеокурсы | YouTube (unlisted) / Vimeo | Надёжный хостинг, встраивается в Tilda |
| Платежи | Stripe | Международные карты |
| Telegram-бот | Node.js + Telegraf | Тесты, автопроверка, результаты |
| Живые занятия | Zoom | Вручную, без автоматизации |
