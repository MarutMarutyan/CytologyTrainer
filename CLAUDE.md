# CytologyTrainer — Правила для Claude

## О проекте

Образовательная платформа по клинической цитологии.
Подробнее: `docs/project.md`

## Стек

- **Сайт:** Tilda (no-code)
- **Видео:** YouTube unlisted / Vimeo
- **Платежи:** Stripe
- **Telegram-бот:** Node.js + Telegraf (папка `telegram-bot/`)
- **Живые занятия:** Zoom вручную

## Правила разработки

- Коммитить после каждой рабочей фичи бота
- Весь код бота — в папке `telegram-bot/`
- Не добавлять другие виды цитологии (кроме гинекологии и щитовидной железы)
- Не делать мобильное приложение, форум, автосертификаты
- Хранить тесты и вопросы в `telegram-bot/src/data/` (JSON-файлы)
- Никогда не коммитить `.env` файлы

## Структура бота

```
telegram-bot/
├── src/
│   ├── index.js        ← точка входа
│   ├── handlers/       ← обработчики команд
│   └── data/           ← тесты и вопросы (JSON)
├── package.json
└── .env.example
```

## Команды запуска

```bash
cd telegram-bot
npm install          # первый раз
npm start            # запустить бота
```
